Test Case Design

In here , we will demonstrate the testing of the andriod with some use cases . The testing is done in 2 phases :

  • * Black Box Testing
  • * White Box Testing

    • Black Box testing : In this method , as the name itself suggests , we will be testing the application without knowing the internal structure or components present in it .We just provide different inputs and check whether we are getting outputs as per our calculations and predictions .


    White Box Testing : This method is almost opposite of black box testing wherein the person or the user testing will be fully aware of the inner workings,design and components of the application.
