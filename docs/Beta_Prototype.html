Testing Process and Test Design Techniques

There are several testing processes which require to be done in parallel or different stages and several techniques inheret in. Test dsign techniques can be experimental based, structred based or specification based. While specification based is functional and blackbox procedure will be applied to and structure based conforms to white box procedure.

Black Box testing:
Using this approach, as the name itself suggests, we will be testing the application without knowing the internal structure or components present in it .We just provide different inputs and check whether we are getting outputs as per our calculations and predictions.

White Box Testing:
This method is almost opposite of black box testing wherein the person or the user testing will be fully aware of the inner workings,design and components of the application.
